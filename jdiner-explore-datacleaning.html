The following function creates a new column in the dataset putting the Death Chapter (or Intro Chapter) across books in the same timeline. For example, if one character dies on episode 6 of the 2nd book; it's real Death Chapter on the GoT timeline is 72+6=78

In [12]:
def chapter_norm(row, column):
  if column == 'Dead':
    if pd.isnull(row.Book_of_Death): #Character is Alive
        row['Death_Chapter_Norm'] = row.Death_Chapter #Copy NaN
        return row
    else:  #Character is Dead
        book = int(row.Book_of_Death)
        chapter = row.Death_Chapter
        if pd.isnull(chapter): #In case the character died but no chapter is listed, assign the last chapter of that book
          chapter=max_chapter[book-1]
        else:
          chapter = int(chapter)
        for i in range(book-1):
          chapter += max_chapter[i]
    row['Death_Chapter_Norm'] = chapter
  elif column == 'Intro':
    if row['GoT']==str(1):
      book = 1
    elif row['CoK']==str(1):
      book = 2
    elif row['SoS']==str(1):
      book = 3
    elif row['FfC']==str(1):
      book = 4
    else:
      book = 5
    chapter = row.Book_Intro_Chapter
    if pd.isnull(chapter): #In case no chapter is listed, assign the first chapter of that book
      chapter=1
    else:
      chapter = int(chapter)
    for i in range(book-1):
      chapter += max_chapter[i]
    row['Intro_Chapter_Norm'] = chapter
  return row

There were inconsistencies in the way the Allegiances were represented, as some rows had the prefix 'House ' and some didn't. Because they represent the same house, we eliminate this prefix

def house_norm(row):
    if row.Allegiances.find('House')==0: #0 if its in the string, -1 if its not
      #Contains House in the name
      row['Allegiances']=row.Allegiances.replace('House ','')
    return row

We want to calculate how many chapters a character was alive, that is, how many chapters passed in the timeline since it was first introduced and killed.

In [14]:
def chapters_alive(row):
  if pd.isnull(row.Book_of_Death): #Is Alive
     row['ChaptersAlive'] = total_chapter-int(row['Intro_Chapter_Norm'])
  else:
    row['ChaptersAlive'] = int(row['Death_Chapter_Norm'])-int(row['Intro_Chapter_Norm'])
    if row['ChaptersAlive']<0: #If chapters alive is negative, cap it to 0
      row['ChaptersAlive']=0
  return row

def gender_norm(row):
  if row.Gender == '1':
    row['Gender'] = 'Male'
  else:
    row['Gender'] = 'Female'
  return row

Putting all the functions together in one function¶

In [16]:
def clean_data(row):
    row = chapter_norm(row,'Dead')
    row = house_norm(row)
    row = chapter_norm(row,'Intro')
    row = chapters_alive(row)
    row = gender_norm(row)
    return row
